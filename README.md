这里是工作和学习中积累的一些C语言函数。将会随着工作和学习的进行，不断补充。

目录说明: 

dep/  是依赖的一些第三方的库（每个文件依赖哪个库，以及每个库被哪个文件依赖，都在各自的注释或目录下README.md中有说明）（每个第三方库在dep目录下独自拥有一个目录）

include/  ljac提供的头文件，make的时候会把ljac对外提供的头文件复制到这里

lib/  ljac编译得到的连接库文件，当前只有静态库libljac.a

文件说明:

ljapkt.[hc]  对原始网络数据包的解析

ljaplatform.h 在使用ljac的时候，需要根据自己的平台情况设置这个文件（字长，大小端等）

ljatype.h  最基本数据类型的封装，在ljac中全部使用这里定义的基本数据类型（第三方库除外）

ljapcap.[hc]  对pcap的一些简单封装，依赖pcap库

ljabits.[hc] 按位操作
